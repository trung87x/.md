## (Trạng thái tương tác)

Kỹ năng: Xử lý hiệu ứng khi người dùng di chuột (hover), nhấn giữ (active), hoặc chọn vào ô nhập liệu (focus) mà không cần viết một dòng CSS nào khác.

```html
<button
  class="bg-indigo-500 text-white px-6 py-2 rounded-full 
               transition-all duration-300
               hover:bg-indigo-700 hover:shadow-lg 
               focus:outline-none focus:ring-4 focus:ring-indigo-300
               active:scale-95"
>
  Mua ngay
</button>
```
