## (Mobile-first)

Kỹ năng: Mặc định viết cho điện thoại trước, sau đó dùng các tiền tố md:, lg: để mở rộng cho màn hình lớn. Đây là kỹ năng sống còn để trang web "bán được tiền".

```html
<div class="flex flex-wrap gap-4">
  <div class="w-full md:w-1/2 lg:w-1/3 bg-blue-500 p-6 text-white rounded">
    Card linh hoạt theo màn hình
  </div>
</div>
```
