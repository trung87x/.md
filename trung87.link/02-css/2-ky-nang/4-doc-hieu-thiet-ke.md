## (Pixel Perfect)

Kỹ năng: Chuyển hóa các thông số từ Figma (như mã màu Hex, độ dày font, khoảng cách) thành CSS chuẩn xác.

Thông số từ Figma: Color: #FF5733, Font-size: 18px, Letter-spacing: 0.5px.

Chuyển hóa sang Code:

```css
.hero-title {
color: #ff5733; /_ Mã màu chính xác từ thiết kế _/
font-size: 1.125rem; /_ = 18px / 16px (đơn vị linh hoạt) _/
letter-spacing: 0.031rem; /_ Tạo độ thoáng cho chữ theo bản vẽ _/
line-height: 1.5; /_ Khoảng cách dòng giúp dễ đọc _/
}
```
